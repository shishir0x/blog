## Quick Start

The core function of Easy Split is to help you:
1. **Take a photo of a receipt** → Automatically recognize bill items
2. **Select participants** → Decide who needs to share the bill
3. **Allocate items** → Tell the system which ones are "Private" (borne by yourself only) and which are "Shared" (split equally)
4. **View totals** → See exactly how much everyone needs to pay at a glance

The entire process is **synchronized in real time**, and everyone sees the latest data!

## Step 5: Allocate Expenses

This is the core feature of Easy Split! You can decide whether each bill item is "Private" or "Shared".

### 5.1 Understand Allocation Methods

Easy Split supports two allocation methods:

#### Private Items (Private)
- **Meaning**: This item belongs to one person only, and the cost is entirely borne by this person
- **Example**: A coffee you bought for yourself, a book a friend bought alone
- **Operation**: Click the "Claim private" button

#### Shared Items (Shared)
- **Meaning**: This item is shared by everyone, and the cost is split equally
- **Example**: A pizza eaten together, shared drinks
- **Operation**: Do nothing, default is shared

### 5.2 Claim Private Items

<ImageSetter src="https://raw.githubusercontent.com/alvinluo-tech/alvin-luo-assets/main/easy-split/05-02-claim-item-button.png" alt="Claim Item" caption="Claim Private Item Button" />

If you are a participant in the bill, you can claim private items:

1. In the "Items" list, find the item you want to claim
2. Click the "Claim private" button
3. The button will change to your name, indicating that the item has been claimed by you
4. The cost will automatically be removed from shared expenses and included only in your personal expenses

<ImageSetter src="https://raw.githubusercontent.com/alvinluo-tech/alvin-luo-assets/main/easy-split/05-02b-item-claimed-by-user.png" alt="Claimed Item" caption="Item has been claimed by user" />

> 💡 **Tip**:
> - Only participants can claim items
> - You can unclaim an item to make it shared again
> - If an item has already been claimed by someone else, you cannot claim it

### 5.3 Edit Item Price

<ImageSetter src="https://raw.githubusercontent.com/alvinluo-tech/alvin-luo-assets/main/easy-split/05-03-edit-item-price.png" alt="Edit Price" caption="Edit Bill Item Price" />

If the AI-recognized price is inaccurate, the bill creator can manually modify it:

1. Find the item that needs modification
2. Click the "Edit" button next to the price
3. Enter the correct price
4. Click the "Save" button
5. The system will automatically update the totals

> 💡 **Tip**:
> - Only the bill creator can edit prices
> - Once the price is modified, all related calculations will update automatically

## Step 6: View Totals

The totals page displays the amount everyone should pay, including private and shared expenses.

### 6.1 View Totals

<ImageSetter src="https://raw.githubusercontent.com/alvinluo-tech/alvin-luo-assets/main/easy-split/06-01-totals-summary.png" alt="Totals Display" caption="Bill Totals Summary" />

At the bottom of the bill details page, you will see the "Totals" section:

1. **Shared subtotal**: The total amount of all shared items
2. **Personal details**: The detailed expenses of each participant
   - **Private**: The total amount of private items you claimed
   - **Share**: Your portion of the shared expenses
   - **Total**: The total amount you need to pay

### 6.2 Understand Expense Calculation

Expense calculation formula:

```
Personal Total = Private Expenses + Shared Expenses

Where:
- Private Expenses = Sum of all private item prices you claimed
- Shared Expenses = Total Shared Amount ÷ Number of Participants
```

**Example**:

Suppose a bill has the following items:
- Pizza: £20 (Shared)
- Drinks: £10 (Shared)
- Coffee: £5 (Claimed by you as private)

Participants: 3 people (You, Friend A, Friend B)

Calculation process:
- Total shared amount: £20 + £10 = £30
- Each person's share: £30 ÷ 3 = £10
- Your private expenses: £5
- **Your Total**: £10 + £5 = £15

### 6.3 Real-time Updates

An important feature of Easy Split is **real-time synchronization**:

- When someone claims or unclaims an item, everyone's totals update instantly
- When the bill creator adds or removes participants, shared expenses are recalculated automatically
- When a price is modified, all related calculations update automatically

You don't need to refresh the page; all changes appear in real time!

## FAQ

### Q1: What if the receipt recognition is inaccurate?

**A**: If the AI-recognized item name or price is inaccurate, you can:
- The bill creator can manually edit the item price
- Item names currently cannot be edited, but you can correct the allocation by claiming the item

### Q2: Can I join multiple communities?

**A**: Yes! You can join multiple communities simultaneously; each community is independent and does not affect the others.

### Q3: What if I accidentally claimed the wrong item?

**A**: You can click the already claimed item again to unclaim it, reverting it to a shared item.

### Q4: How are shared expenses calculated?

**A**: Shared Expenses = Total amount of all shared items ÷ Number of participants. The system distributes it equally automatically.

### Q5: Can I change the community name?

**A**: Yes! If you are the owner, you can click the edit button next to the community name on the community page to change it.

### Q6: Will the invite code expire?

**A**: No. The invite code is permanent and remains valid as long as the community exists.

### Q7: What if I didn't receive the verification email?

**A**: Please check:
- Spam folder
- Whether the email address is correct
- You can click "Resend verification email" to try again

### Q8: Can I delete a community?

**A**: Currently, deleting a community is not supported. If you are the owner, you can remove all members, but the community itself cannot be deleted.

### Q9: How long is bill data saved?

**A**: Bill data is saved permanently until you manually delete it.

### Q10: Which currencies are supported?

**A**: Currently, bills are based in GBP (British Pound), and you can manually input an exchange rate to display the equivalent CNY (Chinese Yuan) amount.
